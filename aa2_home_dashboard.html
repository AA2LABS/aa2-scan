HOME DASHBOARD — ARCHITECTURE DOCTRINE · PRE-BUILD LOCK
FIRST SCREEN AFTER ONBOARDING
This dashboard is where the app lands after SEAL THE MEMBRANE. Not the scanner. The dashboard. The scanner is a tool. The dashboard is the nervous system made visible.
FULL STACK vs PHONE ONLY
Full stack shows biosignal baseline card with 90-day averages, equalizer bands, hardware connected list. Phone only shows scanner grid, recent scans, Act Right — and an upgrade prompt where the baseline card would be.
THE EQUALIZER BANDS
10 bands per member. Color: green = above baseline, gold = at or below baseline. Center line = YOUR baseline — never a population average. This is the Dashboard Visual Doctrine from Canon v25.
BASELINE DATA SOURCE
Reads from Supabase baseline_profiles: sleep_score, hrv_baseline_90d, readiness_baseline_90d, resting_heart_rate. Populated by hardware upload at onboarding. If null — shows upgrade prompt.
RECENT SCANS
Last 3 scans from scan history. Shows product name, tab, time, location if available, verdict dot color-coded. SEE ALL links to full scan history. Both tiers get this.
ACT RIGHT DOLLARS
Both tiers earn Act Right Dollars. Phone-only member at $12. Full stack member at $47. The debit card is visible on both. The balance grows with every smart swap — hardware accelerates it but doesn't gate it.
